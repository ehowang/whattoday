# Browser Language i18n (English + Chinese)

## Detection
Read `navigator.language` client-side. Starts with `zh` → Chinese, otherwise → English.

## Approach
Simple dictionary object in `src/lib/i18n.ts`. No i18n library. `useLocale()` hook + `t(key)` function.

## Strings (~25)
- Landing: title, description, create button
- Slot machine: empty state, paytable label, button labels, status
- Result: jackpot, result header, dismiss text
- Drawer: title, empty state, form labels
- Card: confirm/deny
- Client: share, copied, aria labels
- Metadata: page title + description

## Not translated
Paytable emoji combos and numbers (universal).
